# Envidicy Dashboard

Standalone analytics SaaS project extracted from Envidicy.

## Goal

Build a dedicated dashboard product with three access layers:

- Envidicy platform layer
- Agency admin layer
- Agency client viewer layer

## Local

```bash
cd /Users/main/Projects/envidicy-dashboard
npm install
npm run dev
```

## Current base

The current implementation uses the existing Envidicy dashboard UI as a foundation and will evolve toward:

- multi-tenant agency model
- provider integrations
- sync pipeline
- reporting/export center
- conversion API ingestion
